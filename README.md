# Parth Dixit - Enterprise AI Portfolio

Welcome to the source code for my personal portfolio! This project is designed as an interactive, high-performance web experience tailored to showcase my work in Artificial Intelligence, Machine Learning, and Data Science.

## 🚀 Features

- **Immersive 3D Background:** A neural network-inspired interactive particle system powered by `Three.js`.
- **Custom AI Chatbot:** A built-in floating AI assistant that can guide visitors through my projects, skills, and contact information.
- **Dynamic Animations:** Smooth scroll triggers and timeline animations utilizing `GSAP` (GreenSock Animation Platform).
- **Data Visualizations:** Interactive charts built with `Chart.js` to represent complex analytics and predictive forecasting data.
- **Glassmorphic UI:** A premium, dark-themed user interface utilizing modern CSS techniques like backdrop-filters and custom gradients.
- **Fully Responsive:** Adapts seamlessly to all screen sizes, ensuring a flawless experience on both desktop and mobile devices.

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid)
- **Logic:** Vanilla JavaScript (ES6+)
- **Libraries:**
  - [Three.js](https://threejs.org/) (3D Graphics & Particles)
  - [GSAP](https://gsap.com/) (Advanced Scroll Animations)
  - [Chart.js](https://www.chartjs.org/) (Interactive Data Visualization)
  - [Typed.js](https://github.com/mattboldt/typed.js/) (Terminal typing effects)
  - [Vanilla Tilt.js](https://micku7zu.github.io/vanilla-tilt.js/) (3D Card Hover Effects)

## 🚀 Getting Started

Since this is a vanilla frontend project, no complex build tools or package managers are required!

1. Clone the repository:
   ```bash
   git clone https://github.com/Parth829/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in your favorite web browser, or use an extension like VS Code's **Live Server** to run it locally.
